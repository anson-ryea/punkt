Punkt (Swedish for *dot*) is a lightweight dotfile manager designed with simplicity in mind. 

It is written in Kotlin and builds native executables with GraalVM for fast, self-contained distribution.
<img src="https://github.com/user-attachments/assets/6b323771-b143-4106-bf87-684782ff5fcf" width="200" />

Punkt is in the early stages of development and may change as features are added and refined.
